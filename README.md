# FUNDR: Credit Risk Prediction & Interactive Simulator

FUNDR is an enterprise-grade machine learning project designed to predict loan default risks using historical LendingClub data. It features a hyper-parameter tuned XGBoost model and a beautiful, interactive "glassmorphism" simulator dashboard that allows users to adjust financial variables and see real-time probability changes.

## Features

- **Massive Dataset**: Trained on 1.3 million historical loan records.
- **Deep Feature Engineering**: Utilizes over 25 granular credit variables and engineers custom financial ratios (e.g., Loan-to-Income and Payment-to-Income) for peak predictive accuracy (0.724 pre-funding ROC-AUC).
- **Auto-Fetch Dataset Mechanism**: Handles dataset loading from Google Drive dynamically to bypass GitHub file size limits.
- **Interactive Browser UI**: A fully local `dashboard.html` interface that simulates logistic regression probabilities directly in Javascript using weights extracted from the Python backend.

## Setup & Installation

**1. Create a Virtual Environment**
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Configure Dataset Download**
Due to the massive size of the dataset, it is not stored in source control. It is hosted on Google Drive. 
- Open `scripts/fetch_data.py`
- Find the variable `DRIVE_FILE_ID = "YOUR_GOOGLE_DRIVE_FILE_ID_HERE"`
- Replace it with the File ID of your Google Drive dataset.

## How to Run

**1. Train the Model**
Run the main XGBoost script. If the dataset isn't found locally, the script will automatically use `gdown` to download it from Google Drive first.
```bash
python ml_pipeline/train_xgboost.py
```
*(Note: Training runs a deep RandomizedSearchCV on 1.3 million rows with 25+ features. This may take 5-10 minutes to complete depending on your hardware).*

**2. Start the Backend Server**
The interactive simulator is served via a FastAPI backend. Start it with:
```bash
python main.py server
```

**3. View the Simulator**
Once the server is running, the interface will be available locally.
- Open `http://127.0.0.1:8000/#drift` in your web browser.
- Scroll down to the Interactive Simulator and adjust the sliders to see real-time risk predictions!
